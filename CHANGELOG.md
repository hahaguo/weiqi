# 版本更新日志

## v1.1.0 (2026-02-06)

### 🎮 新功能

**人机对弈系统**
- ✨ 添加完整的人机对弈功能
- 🤖 三种AI难度级别：简单、中等、困难
- 🎯 基于启发式规则的AI引擎
- 💡 智能评估系统（救子、提子、防守、连接、切断、围地等）

**用户界面**
- 🎨 新增AI设置面板，可选择难度和先后手
- 💫 AI思考动画和进度显示
- 📊 显示AI落子策略和理由
- 🌈 精美的紫色渐变主题

**技术改进**
- ⚡ 性能优化：智能过滤候选位置
- 🔧 模块化架构：AI逻辑独立封装
- 🔌 可扩展设计：预留云端AI接口
- 📦 完整的服务抽象层

### 📁 新增文件

**AI核心模块**
- `src/ai/goAI.js` - AI引擎核心
- `src/ai/aiEvaluators.js` - 评估函数库
- `src/api/aiService.js` - AI服务抽象层

**UI组件**
- `src/components/AISettings.jsx` - AI设置面板
- `src/components/AISettings.css` - 设置面板样式
- `src/components/AIGame.jsx` - 人机对弈游戏组件
- `src/components/AIGame.css` - 游戏界面样式

### 🔄 修改文件

- `src/goLogic.js` - 添加AI辅助方法（cloneGame、getAllEmptyPoints、getGroupInfo）
- `src/App.jsx` - 集成人机对弈模式和路由
- `package.json` - 更新版本号到1.1.0
- `docker-compose.yml` - 更新版本标签

### 🎯 AI难度说明

- **简单** 😊：30%随机性，思考0.5秒，基础战术
- **中等** 🤔：15%随机性，思考1秒，中级策略
- **困难** 😈：5%随机性，思考2秒，完整规则

### 🚀 使用方法

1. 主菜单点击"🤖 人机对弈"
2. 选择AI难度和执子颜色
3. 点击"开始对弈"
4. 享受与AI的对弈！

### 🐛 已知问题

- 云端AI功能尚未实现（预留接口）
- 19×19大棋盘时AI思考可能略慢（已优化）
